# portal-cns-journal

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存放，便于浏览、查阅和二次分发。

## 项目简介

portal-cns-journal 是一个纯前端静态页面集合。所有页面均为独立 HTML 文件，不依赖后端服务，可直接在浏览器中打开或部署到任何静态托管平台。

## 目录说明

```
/
├── index.html          # 仓库主页（可选）
├── pages/              # 归档页面目录
│   ├── example.html
│   └── ...
└── assets/             # 共用资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：存放所有归档的 HTML 页面，每个文件为一个独立页面。
- `assets/`：存放页面共用的静态资源文件，按类型分目录存放。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可脱离本仓库单独使用。
- 页面内容不针对特定网站或域名，仅作为通用信息展示或功能演示。
- 归档的页面可能包含演示数据、示例文案或占位内容，请勿直接用于生产环境。
- 部分页面可能引用 `assets/` 下的资源，迁移时需注意保持路径一致。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来改进现有页面或新增归档。
- 新增页面请放置在 `pages/` 目录下，并尽量保持资源引用规范。
- 请勿在页面中嵌入任何形式的跟踪脚本、广告或外部请求（除 CDN 资源外）。
- 本仓库不承诺长期维护，但会尽力保持页面可访问。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 下的 HTML 文件。
3. 或者将整个仓库部署到任意静态托管服务（如 GitHub Pages、Netlify 等）。
